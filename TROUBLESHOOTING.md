# راهنمای رفع مشکلات (Troubleshooting)

## مشکل: Timeout Error

اگر ربات با خطای "Timed out" متوقف می‌شود، این معمولاً به این معنی است که نمی‌تواند به سرورهای تلگرام متصل شود.

### راه حل‌ها:

#### 1. استفاده از VPN یا Proxy

اگر در کشوری هستید که تلگرام فیلتر است (مانند ایران)، باید از VPN یا Proxy استفاده کنید:

1. یک VPN فعال کنید که به سرورهای تلگرام دسترسی دارد
2. یا از یک HTTP Proxy استفاده کنید

#### 2. تنظیم Proxy در کد

اگر می‌خواهید از Proxy استفاده کنید:

1. فایل `bot/config.py` را باز کنید
2. خط زیر را پیدا کنید:
   ```python
   PROXY_URL = None  # Set to None if no proxy needed
   ```
3. آن را به این صورت تغییر دهید:
   ```python
   PROXY_URL = "http://proxy.example.com:8080"  # آدرس proxy خود را قرار دهید
   ```

#### 3. بررسی اتصال اینترنت

- مطمئن شوید که اینترنت شما فعال است
- تست کنید که آیا می‌توانید به وب‌سایت‌های دیگر دسترسی داشته باشید

#### 4. بررسی فایروال

- مطمئن شوید که فایروال شما مانع اتصال نمی‌شود
- ممکن است نیاز باشد Python را به لیست استثنائات فایروال اضافه کنید

#### 5. بررسی توکن ربات

- مطمئن شوید که توکن ربات شما معتبر است
- می‌توانید با دستور زیر تست کنید:
  ```bash
  python test_telegram_simple.py
  ```

### تست اتصال

برای تست اتصال به تلگرام:

```bash
python test_telegram_simple.py
```

اگر این تست هم timeout داد، مشکل از اتصال اینترنت یا فیلترینگ است.

### نکات مهم

- اگر از VPN استفاده می‌کنید، مطمئن شوید که قبل از اجرای ربات، VPN را فعال کرده‌اید
- Proxy باید از نوع HTTP باشد
- برخی VPN ها ممکن است نیاز به تنظیمات اضافی داشته باشند
